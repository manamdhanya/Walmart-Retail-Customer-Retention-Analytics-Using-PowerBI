# Walmart-Retail-Customer-Retention-Analytics-Using-PowerBI

## 📌 Description

This project analyzes **customer retention and churn behavior in Walmart retail data** using data analytics techniques. The goal is to understand customer purchasing patterns, evaluate loyalty programs, measure churn rates, and identify strategies to improve customer retention.

The project includes **data cleaning, churn analysis, customer segmentation, promotion effectiveness analysis, and customer lifetime value (CLV) evaluation**.

## 🎯 Objectives

* Analyze customer churn and retention patterns
* Identify high-value and repeat customers
* Evaluate the effectiveness of promotions and loyalty programs
* Study store and channel performance
* Calculate Customer Lifetime Value (CLV)
* Provide business recommendations to improve customer retention

## 🛠 Tools Used

* Power BI
* Power Query
* DAX
* Data Modeling (Star Schema)
* Data Visualization

## 📊 Dataset Overview

The analysis uses multiple datasets:

* Customer Demographics
* Customer Transactions
* Churn Labelled Customers
* Loyalty Program
* Store Locations

These datasets were integrated using a **star schema data model** for efficient analysis.

## 🔎 Key Analysis Performed

### Data Preprocessing

* Cleaned and standardized datasets using Power Query
* Converted fields to appropriate data types
* Created calculated columns for membership duration and store age

### Churn and Retention Analysis

* Calculated KPIs such as:

  * Total Customers
  * Churned Customers
  * Churn Rate
* Analyzed churn across:

  * Regions
  * Income groups
  * Loyalty tiers
  * Customer channels

### Repeat Purchase Analysis

* Segmented customers based on purchase frequency:

  * Low Tier (0–3 purchases)
  * Mid Tier (4–8 purchases)
  * High Tier (9+ purchases)
* Compared purchase behavior across demographics and loyalty tiers

### Promotion and Loyalty Analysis

* Evaluated promotion effectiveness
* Compared transactions with and without promotions
* Studied engagement using loyalty points earned and redeemed

### Store and Channel Performance

* Compared store types based on transaction value and churn
* Analyzed performance of online vs physical store channels
* Studied relationship between store age and retention

### Customer Lifetime Value (CLV)

* Calculated total amount spent by each customer
* Derived CLV based on membership duration
* Segmented customers into **High CLV and Low CLV groups**

## 📈 Key Insights

* Churn varies significantly across regions and loyalty tiers
* Promotional campaigns do not significantly increase transaction value
* Loyalty engagement affects customer retention
* Some channels and store types underperform in retention metrics

## 📊 Dashboard Components

The final dashboard includes:

* Churn and retention KPIs
* Customer segmentation analysis
* Loyalty and promotion insights
* Store and channel performance analysis
* Customer lifetime value analysis

## 📢 Business Recommendations

* Target **high-CLV inactive customers** to reduce churn
* Improve performance of **online and neighborhood market channels**
* Strengthen loyalty program engagement
* Provide incentives for low-frequency customers
